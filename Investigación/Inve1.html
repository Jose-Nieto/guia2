    • Funciones del metodo Math

    El método Math.abs () se usa para devolver el valor absoluto de un número. Toma un número como parámetro y devuelve su valor absoluto.

    La función Math.round() es muy sencilla ya que al tener un numero de tipo decimal retorna el valor de un número redondeado al entero más cercano.

    La función Math.ceil() es igual de sencilla, lo que hace es que devuelve el entero mayor o igual más próximo a un número dado.

    La función Math.floor() lo que hace es que devuelve el máximo entero menor o igual a un número.

    La función Math.exp() devuelve ex, donde x es el argumento, y e es El número de Euler (también conocido como la constante de Napier), la base de los algoritmos naturales.

    La función Math.log() devuelve la base neutral de un número (base e), es una funcion del metodo Math y es muy ocupada

    La función Math.random() retorna un punto flotante, un número pseudo-aleatorio dentro del rango [0, 1). Esto es, desde el 0 (Incluido) hasta el 1 pero sin incluirlo (excluido), el cual se puede escalar hasta el rango deseado.

    Metodo Pop y Push

    El método pop() elimina el último elemento de un array y lo devuelve. Este método cambia la longitud del array.

    El método push() añade uno o más elementos al final de un array y devuelve la nueva longitud del array.

    Reverse

    El método reverse() invierte el orden de los elementos de un array in place. El primer elemento pasa a ser el último y el último pasa a ser el primero.
